## How to Start

1.  **Use this template**: Click the "Use this template" button to create a new repository from this template.
2.  **Enable Actions**: Go to the "Actions" tab in your new repository and enable workflows if they are disabled.
3.  **Start the Course**: The course will start automatically when you push to the `session1-enablement` branch or you can manually trigger the "Start Exercise" workflow from the Actions tab.
4.  **Follow the Issues**: The course instructions will be provided via GitHub Issues. Check the "Issues" tab for your first task!
